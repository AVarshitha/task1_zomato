# Zomato Dataset Analysis
This repository contains the data cleaning, exploratory data analysis (EDA), and visualizations for the Zomato restaurant dataset as part of the Alfido Tech internship program.

## Task Objective
To systematically analyze restaurant profiles, geographic densities, pricing structures, and core trends driving user ratings.

## Technology Stack
* Python
* Google Colab
* Pandas & NumPy
* Matplotlib & Seaborn
* WordCloud
